# MyFavSongsAPI

MyFavSongsAPI is a small, full-featured system designed to allow clients to create, manage, and organize their favorite songs, albums, and artists. The project provides a web API for managing these entities and stores all information in an SQLite database.

## Features

- Create, read, update, and delete (CRUD) operations for:
  - Songs
  - Albums
  - Artists
- API endpoints for easy integration with other services or front-end applications.
- Lightweight and efficient storage using SQLite.
